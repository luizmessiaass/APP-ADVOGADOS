The workflow MUST:
1. Trigger on: push to main/master branches + pull_request to main/master
2. Run on: ubuntu-latest
3. Set up JDK 17 (temurin distribution) — required for Android SDK 36 build
4. Cache Gradle dependencies and build cache (key: ${{ runner.os }}-gradle-${{ hashFiles('**/*.gradle*', '**/libs.versions.toml') }})
5. Run in sequence:
   a. `./gradlew :app-cliente:lintDemoDebug` — zero lint errors gate
   b. `./gradlew :app-cliente:test` — unit tests gate (SyncLabelFormatterTest, ViewModelTests)
   c. `./gradlew :app-cliente:assembleDemoDebug` — build verification

Full workflow YAML (create or replace app-cliente steps):

```yaml
name: Android CI

on:
  push:
    branches: [ main, master ]
  pull_request:
    branches: [ main, master ]

jobs:
  build-app-cliente:
    name: app-cliente lint + test + build
    runs-on: ubuntu-latest

    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Set up JDK 17
        uses: actions/setup-java@v4
        with:
          java-version: '17'
          distribution: 'temurin'
          cache: gradle

      - name: Cache Gradle packages
        uses: actions/cache@v4
        with:
          path: |
            ~/.gradle/caches
            ~/.gradle/wrapper
          key: ${{ runner.os }}-gradle-${{ hashFiles('**/*.gradle*', '**/gradle-wrapper.properties', '**/libs.versions.toml') }}
          restore-keys: |
            ${{ runner.os }}-gradle-

      - name: Grant execute permission for gradlew
        run: chmod +x gradlew

      - name: Lint app-cliente
        run: ./gradlew :app-cliente:lintDemoDebug

      - name: Unit tests app-cliente
        run: ./gradlew :app-cliente:test

      - name: Assemble app-cliente demo debug
        run: ./gradlew :app-cliente:assembleDemoDebug

      - name: Upload lint report
        if: always()
        uses: actions/upload-artifact@v4
        with:
          name: lint-results-app-cliente
          path: app-cliente/build/reports/lint-results-demoDebug.html

      - name: Upload test results
        if: always()
        uses: actions/upload-artifact@v4
        with:
          name: test-results-app-cliente
          path: app-cliente/build/test-results/
```

If an existing workflow already has an `app-escritorio` job, keep it and add `build-app-cliente` as a separate job (they run in parallel by default in GitHub Actions).

IMPORTANT: Do NOT add `./gradlew :app-cliente:connectedDemoDebugAndroidTest` to the CI workflow — instrumented tests require an emulator which is slow and flaky in CI (per Phase 0 D-20 pattern).
